# Paradise Nursery Shopping Application (e-plantShopping)

## Description
Paradise Nursery is a dynamic web application for an online plant shop. It allows users to browse a variety of houseplants, view details, add them to a shopping cart, and manage their cart items seamlessly.

## Features
- Interactive product listing with multiple categories.
- Dynamic shopping cart using Redux for state management.
- Calculate total costs and update item quantities dynamically.
- Responsive navigation bar.

## Usage Instructions
1. Click "Get Started" on the landing page.
2. Browse plants by category (Air Purifying, Low Light, Pet Friendly).
3. Click "Add to Cart" to select a plant.
4. Navigate to the Cart to update quantities, remove items, or proceed to checkout.
